# Lovefire

This is a compatibility site that combines zodiac signs × blood types × MBTI × 16 love types diagnosis.

## Features
- Instant diagnosis from 12,288 combinations
- Displays BEST 5 (highest compatibility) and WORST 5 (lowest compatibility)
- Responsive design (works on smartphone)
- Remembers your type using session

## How to Use
1. Select your zodiac sign, blood type, MBTI, and love type
2. Click "診断する" (Diagnose)
3. See your compatibility rankings!

## Tech Stack
- Backend: Flask (Python)
- Frontend: Jinja2 + Custom CSS
- Deployment: Render.com (planned)
